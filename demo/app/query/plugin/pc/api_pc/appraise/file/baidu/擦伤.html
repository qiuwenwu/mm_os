症状识别
1、观察伤口大小
查看创面的大小、深度与清洁程度；
2、观察出血情况
查看伤处有无血痕、渗血或出血斑点，并触摸创面周围皮肤，观察其是否有压痛、畸形并伴有腕关节活动受限等情况。如果伴有畸形或腕关节活动受限等情况，请按手腕摔伤进行处理。
救治方法
1、若伤者创面较小，伤口干净且不深
可先用清水对创面进行清洗，再由内向外涂抹碘伏，消毒2次即可（对创面不要进行包扎）；
2、若伤者创面较小，伤口较浅，且沾有泥土、沙粒等污物时
需先用肥皂水冲洗创面，再用清水将创面残留的肥皂水冲洗干净，并擦干。最后在创面由内向外涂抹碘伏消毒2次，并任其自然干燥即可。
特别提示
1、禁止涂抹牙膏、药膏、草药、粉末等物质，以免掩盖伤者伤情，造成感染；
2、糖尿病伤者无论创面大小，都应及时到医院就诊；
3、避免伤口受到不必要的损伤，如受压、摩擦或敷料过敏等；
4、皮肤擦伤后不建议使用创口贴，以免不利于创面分泌物的引流。
