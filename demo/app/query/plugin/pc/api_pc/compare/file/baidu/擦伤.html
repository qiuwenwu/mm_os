擦伤_百度百科

擦伤是钝性致伤物与皮肤表皮层摩擦而造成的以表皮剥脱为主要改变的损伤，又称表皮剥脱，是开放伤中最轻的一类创伤。擦伤多发生于钝器打击、坠落、交通事故等情况。擦伤除伤及表皮外，也可伤及真皮层。擦伤可单独存在，亦可与挫伤、挫裂伤、枪弹创伤，甚至砍伤等并存。

擦伤了怎么办

擦伤了的患者可以用双氧水或生理盐水将擦伤的患处清洗干净，再用碘伏液进行消毒，患者也可以用纱布将患处包扎好以促进恢复。建议擦伤了的患者在恢复期间有注意保养患处，保持擦伤的部位干净卫生，不要沾水，防止细菌感染。在饮食上应避免吸烟喝酒，也不能吃辛辣的刺激性的食物。

李志明主治医师朝阳市第二医院三甲

擦伤一般几天能好

擦伤一般10天左右就可以好了。对于皮肤的擦伤，如果伤口不是很深，表浅的伤口，有一些渗出，可以在表面用碘伏进行消毒，完了以后等待伤口自行结痂，一般10天以后结痂可以自行脱落。但是在这期间不能洗澡，伤口也不能碰水，不然容易导致细菌感染。如果擦伤伤口比较深，有可能表面渗出会多一些，要进行换药，防止伤口内有细菌滋生，引起感染，正常换药情况下10天左右可以好。

汤大纬主治医师合肥市第二人民医院三甲

擦伤怎么处理

检查受伤的部位，一般擦伤、摔伤之类的都是表皮会有好多条划痕，出现红肿、发青的情况，出血的相对较少，如果有量也不会太大，凝血功能正常的人几分钟就止住了。日常的擦伤、摔伤、磕伤大多是浅表型的伤口，其实最好不要用创可贴，让伤口暴露在空气里是无大碍的，在相对的擦一些没激素的复缘膏之类的帮助伤口愈合就行了。

高其宏副主任医师芜湖市第二人民医院三甲

擦伤一般痛几天

通常情况下，擦伤一般会痛2~3天左右，擦伤期间，患者的擦伤部位会出现灼热，跳痛等症状。建议患者要保持擦伤的部位干净卫生，避免沾水，防止感染。建议擦伤的患者，要多注意休息，在饮食上应多吃一些清淡的食物，避免吃辛辣的刺激性的食物，避免吸烟喝酒。

李志明主治医师朝阳市第二医院三甲

擦伤了怎么处理

擦伤属于创伤中最轻微的一种损伤，处理要看面积的大小来决定。如果是轻微的擦伤，面积不大，只要局部进行碘酒消毒，表面涂抹抗生素药膏，保持伤口的干燥，一般一周左右即可恢复；如果是大面积的擦伤，那就必须进行彻底的清创消毒，并且进行包扎治疗，必要时要口服抗生素或静脉注射抗生素。若损伤部位在关节处，还需要进行制动处理。

汤大纬主治医师合肥市第二人民医院三甲

擦伤多久能好呀？

这个都是需要注意好自己的病症表现的，不是很严重的话，大概会在1-2个星期左右会慢慢的愈合，不要着急。伤口要及时消毒清洁，减少膝关节的活动，以免影响到了伤口的恢复速度，而且最好使用无菌纱布包扎。

如何快速正确的处理擦伤-百度经验

1/6

在出现擦伤的情况下，第一步立即清洗伤口，在处理伤口之前先洗手。擦伤会使人的第一道防线——皮肤受损。如果细菌趁机而入，就可能会导致伤口感染。如果出现了感染的情况，你需要去看医生。

2/6

清洗受伤区域。使用干净的毛巾，用温水和抗菌皂，小心的擦拭伤口。擦去所有的脏东西和皮屑，可以预防感染。

3/6

给伤口消毒。你需要给伤口消毒，以免感染。如果伤口处有异物，或者流血的伤口比较深，你可以使用碘酒、双氧水或抗菌剂给伤口消毒。选择一种消毒剂，将棉球浸泡在里面，取出之后轻轻擦拭伤口。消毒过程中可能会引起你的痛感。酒精会引起强烈的刺痛感，对伤口造成损伤，所以不要使用酒精消毒。

4/6

在受伤区域使用抗菌药膏。购买抗菌止痛的药膏，然后涂抹到伤口处。检查伤处，确定是否有更严重的伤口，比如割伤或撕裂伤。如果严重的话，可能需要看医生。

5/6

保持干燥。不要让伤口接触到水，因为潮湿的环境会滋生细菌。如果不小心弄湿了伤口，你可以使用棉球擦干皮肤上的水。如果伤口处有液体渗出，不要随手擦掉它，这可能会刺激到伤口。你可以将纱布或创可贴取下，让它自然风干。如果渗出的是脓液或血，请立即去看医生。

6/6

使用纱布或创可贴。使用创可贴保护伤口。你需要每天更换创可贴，并检查伤口。如果发现伤口已经结疤，那么为了达到最好的愈合效果，你需要停止使用创可贴让伤口直接接触空气。如果伤口没有结疤，你需要更换新的创可贴，然后再等待24小时。

擦伤怎么处理
擦伤的处理，首先受伤后应观察伤口。如果伤口比较大、血流不止，建议立刻就医，不是自己可能解决的问题。如果观察后发现伤口不大、伤口不深、没有大量出血，可以自行处理。第一步，找到流动的水源来冲洗伤口，目的清除伤口内的明显的污物，冲洗完之后用双氧水、碘伏反复的冲洗创口。碘伏及双氧水冲洗完之后，需要再次冲洗创口，这时候可以用蒸馏水，如果蒸馏水很难取得，可以用纯净水。另外，双氧水冲洗过的创口，基本上算理论上无菌伤口，用自来水冲洗反而会增加感染的机会，这次可以用纯净水或者生理盐水进行冲洗。最后一步，将冲洗完以及消毒完的伤口，用无菌纱布进行覆盖，同时每1-2天更换敷料，观察切口愈合情况。如果切口出现了红肿、疼痛、渗液、脓液的情况，仍然需要去医院进行及时的治疗。

皮肤擦伤_百度百科

擦伤是皮肤表面被粗糙物擦破的损伤，最常见的是手掌、肘部、膝盖、小腿的皮肤擦点击此处添加图片说明伤。擦伤后可见表皮破损，创面呈现苍白色，并有许多小出血点和组织液渗出。由于真皮含有丰富的神经末梢，损伤后往往十分疼痛，但表皮细胞的再生能力很强，如伤口无感染则愈合很快，并可不留疤痕。
